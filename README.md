# KLCS_SED_fitting
This repo contains products associated with SED-fitting for KLCS galaxies. SED fitting is conducted using Bagpipes (Carnall et al. 2019), assuming a non-parametric star formation history with the continuity prior from Leja et al. (2019). Since we use photometry that has been corrected for contribution from emission lines, we do not include nebular emission lines in the fit. The stellar population is modelled using BPASS v2.2.1 grids (Stanway & Eldridge, 2018), assuming a Kroupa-like IMF with an upper-mass slope of -2.35 and an upper mass cutoff of 100 solar masses. The full SED-fitting procedure is the same was what is described in Korhonen Cuestas et al. (2025) for KBSS galaxies, and the .h5 files in the posterior contain the full set of fit instructions used.

mass_sfr_age: contains the galaxy ids, log(M), SFR averaged over the past 100 Myr (M_sol/year), mass-weighted ages (Gyr) and associated uncertainties for galaxies in each KBSS field photometry: contains line-corrected multiband photometry for each KBSS field. All given in AB magnitude. posterior: files containing the posterior and fit instructions generated by Bagpipes.
